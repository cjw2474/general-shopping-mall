# general-shopping-mall
스프링 + JSP 종합쇼핑몰 사이트

## 💡 프로젝트 소개
다양한 상품들을 구매할 수 있는 쇼핑몰 사이트 입니다.<br>
<br>

#### 진행 기간
* 21.3.15 - 21.4.2

### ⚙️ 개발 환경
- `Java 8`
- `JDK 1.8.0`
- **IDE** : Eclipse
- **Server** : Apache Tomcat
- **Database** : Oracle DB
