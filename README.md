# general-shopping-mall
스프링 + JSP 종합쇼핑몰 사이트

## 💡 프로젝트 소개
다양한 상품들을 구매할 수 있는 쇼핑몰 사이트 입니다.<br>
<br>

#### 진행 기간
* 21.3.15 - 21.4.2

### ⚙️ 개발 환경
- `Java 8`
- `JDK 14.0.1`
- **IDE** : Eclipse
- **Server** : Apache Tomcat
- **Database** : Oracle DB

## 📌 주요 기능
1. 회원가입
   * 아이디, 비밀번호
   * 아이디 중복검사
2. 로그인
   * 일반 로그인, 관리자 로그인
   * 관리자 권한 기능(상품 등록, 삭제)
3. 마이페이지
   * 개인정보수정
   * 구매 내역
   * 장바구니 페이지
   * 고객센터
   * 문의목록
4. 상품
   * 상품 검색
